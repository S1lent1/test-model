### Что и зачем мы делаем в самом продукте

# Цели

Создание профессиональной соц.сети ориентированной на пользователей связанных с изобразительным искусством, предоставляющей возможность централизованного хранения референсов и проектов, а также интеллектуального поиска материалов для поиска вдохновения и организации карты мыслей для создания собственных проектов. 

# Роли пользователей

#### Default_user
Ориентация на поиск и создание контента для личного пользования

- Возможность создавать доски референсов
- Организация mandmaps для создания нового контента
- Загрузка материалов на личную страницу

#### Content_distributor
Поиск соискателей для создания контента / распространения услуг 

- Возможность создания публичного резюме и вакансии для поиска соискателей
- *Деловой аккаунт* :
  + деловой чат
  + кошелек / связь с кошельками
  + ...
